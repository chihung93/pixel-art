Pixel Art
========
A work in progress pixel art drawing app for Android.

Features
--------
This app currently includes:
 * Simple tools (pen, oval and flood fill)
 * Zooming and panning
 * UI controls (colour palette and a pixel grid)
 * Basic tablet support
 * Plenty of bugs
